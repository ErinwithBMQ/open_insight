

**功能注解**  
1. **项目相关性描述**  
   - 主要功能：提供基于新Bing AI的桌面端聊天应用  
   - 技术定位：无需安装浏览器插件或Microsoft Edge即可实现AI对话交互  
   - 解决问题领域：简化AI聊天体验并支持多平台本地化使用  
   - 关联技术/系统：依赖Microsoft账户认证及代理网络配置（需VPN支持）  

2. **技术栈**  
   - 核心框架/库：未明确提及具体开发框架（如Electron等）  
   - 基础设施/平台：跨平台支持（Windows/macOS/Linux）；需依赖操作系统基础环境及网络协议栈  
   - 关键依赖项：Microsoft账户认证接口；代理服务器配置（需包含`bing.com`及其子域名）  

3. **应用场景**  
   - 主要场景：早期预览版新Bing AI服务访问与本地化交互操作  
   - 适用行业/领域：科技研发、企业级AI工具集成及个人用户日常办公场景  

4. **附加特性**  
   - 支持导出对话记录为Markdown/PNG/PDF格式  
   - 提供主题定制、字体大小调整及快捷键自定义功能（如Ctrl/Cmd组合键控制会话切换与界面操作）
